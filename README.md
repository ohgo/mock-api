# mock-api

[![CircleCI](https://circleci.com/gh/ohgo/mock-api/tree/master.svg?style=svg)](https://circleci.com/gh/ohgo/mock-api/tree/master)

priority: `1`  
method: `GET`  
path: `/`  
response: `{\"statusCode\":200,\"body\":\"hello!\"}`
